# Generate a Feature Guide

1. Create a Product Requirement Document (PRD)

```txt
Use @1.create-PRD.mdc

Here's the feature I want to build: [Describe your feature in detail]
- xxx
- yyy

Reference these files to help you: [Optional: @file1.py @file2.ts]
```

2. Generate Your Task List from the PRD

```text
Now take @PRD-my-feature.md and create tasks using @2.generate-tasks-from-PRD.mdc
```

3. Examine Your Task List

4. Instruct the AI to Work Through Tasks (and Mark Completion)

```text
Please start on task 1.1 in @TASKS-my-feature.md and use @3.process-task-list.mdc
```

5. Review, Approve, and Progress
